# HRain — Offline/Online Mind-Mapping Engine for BCI

Pure manual version of iNaiHR.  
No internet. No API key. No external data transfer.

Designed for real BCI patients (Neuralink, Synchron, Paradromics, Blackrock) where privacy and offline operation are mandatory (HIPAA/GDPR compliance).

### Features
- Full manual control: create, name, connect, delete nodes  
- Same D3.js force-graph physics as iNaiHR  
- All data stored locally in browser (localStorage)  
- Zero external dependencies after first load

https://hawkar-usls.github.io/Hrain/

### How to run
Open `index.html` in any browser — works completely offline.
