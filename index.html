Подсказки HRAIN (ПК/Тачскрин)

  • **Создать узел:** Двойной клик мышью / Двойной быстрый тап пальцем на пустом месте.
  • **Переместить холст (Панорамирование):** Удерживайте ПРОБЕЛ и тяните мышью / Тяните одним пальцем по пустому месту.
  • **Переместить узел:** Тяните мышью / Тяните одним пальцем по узлу.
  • **Зум (Масштаб):** Колесико мыши / Жест **"Pinch-to-Zoom"** (сведение/разведение двух пальцев).
  • **Соединить узлы:** Кликните на первом узле, затем на втором.
  • **Удалить узел/связь:** Двойной клик мышью / Двойной тап пальцем по узлу/связи.
  • **Масштаб узла:** Кнопки **+** и **-** возле узла.
  • **Сохранение:** Происходит автоматически после каждого действия.
